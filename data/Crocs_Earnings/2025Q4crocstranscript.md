# 📈 美股财报高频词汇与商业“黑话”词库 (基于 Crocs 财报精读)

## 一、 核心财务指标与报表术语 (Financial Metrics & Reporting)

| 英文词汇 | 中文释义 | 商业语境与潜台词 |
| --- | --- | --- |
| **GAAP** | 美国通用会计准则 (Generally Accepted Accounting Principles) | 官方的记账标准，所有在美上市公司必须遵守。 |
| **Non-GAAP / Adjusted** | 非公认会计准则 / 经调整的 | 管理层剔除了一次性、非日常开销后的“滤镜版”财务数据，更能反映核心业务的真实赚钱能力。 |
| **Free Cash Flow (FCF)** | 自由现金流 | 扣除维持运转的必要开支后，公司真正可以自由支配的“闲钱”。是衡量公司真实造血能力的核心指标。 |
| **Gross Margin** | 毛利率 | (营收 - 直接成本) / 营收。衡量产品本身暴利程度的核心指标。 |
| **Operating Margin** | 营业利润率 | 毛利扣除日常运营支出(SG&A)后的利润率。衡量公司主营业务整体赚不赚钱。 |
| **SG&A** | 销售、一般及行政管理费用 (Selling, General and Administrative) | 也就是俗称的“日常营业费用”。包括广告营销、直营店开销、高管工资等（注意：美股中不包含财务/利息费用）。 |
| **EPS (Earnings Per Share)** | 每股收益 | 公司净利润除以总股本。直接决定股价走势的终极指标。 |
| **Diluted EPS** | 摊薄后每股收益 | 按“最坏情况”（所有期权等都兑换成股票导致股本变大）计算出的、挤干水分的每股真实收益。 |
| **Balance Sheet** | 资产负债表 | 记录公司在特定时间点的资产、负债和股东权益，看公司的“家底”和抗风险能力。 |
| **Cash and Cash Equivalents** | 现金及现金等价物 | 账上的活期存款以及极容易变现的短期安全资产（如短期国债）。 |
| **Revolver** | 循环信贷 (Revolving credit facility) | 银行批给公司的“巨型信用卡”，随时可借可还，是公司流动性（Liquidity）的强力后盾。 |
| **Effective Tax Rate** | 实际税率 | 公司实际缴纳的税费占税前利润的比例。 |
| **Discrete Costs** | 独立/一次性成本 | 偶尔发生、不具备持续性的开销（常被管理层在算 Adjusted 利润时剔除）。 |

## 二、 业务运营与销售渠道 (Business Operations & Channels)

| 英文词汇 | 中文释义 | 商业语境与潜台词 |
| --- | --- | --- |
| **DTC (Direct to Consumer)** | 直营业务 | 消费者直接在官网或品牌直营店购买。特点是：毛利高、能掌握用户数据。 |
| **Wholesale** | 批发业务 | 把产品卖给第三方经销商或大型商场。特点是：铺货快，但毛利低且容易失去对价格的控制。 |
| **Sell-in** | 向经销商的发货量/铺货量 | 品牌卖给经销商的数量。控制 Sell-in 是为了防止市场上货太多导致打折烂大街。 |
| **Sell-through** | 终端售出率 | 经销商最终卖给真实消费者的数量。 |
| **Inventory Units** | 库存件数/数量 | 仓库里积压的实物数量（区别于库存的美元总价值）。库存下降通常是去化健康的标志。 |
| **Comparable Store Sales (Comp Store)** | 可比门店销售额 / 同店销售额 | 开业一年以上老店的销售对比。衡量品牌真实受欢迎程度（而非靠开新店堆数据）的最关键指标。 |
| **ASP (Average Selling Price)** | 平均售价 | 衡量品牌“定价权”的终极指标。ASP 连年上涨说明消费者愿意花更多钱买，或者买得更贵，公司不需要打折。 |
| **Performance Marketing Spend** | 效果营销支出 | 能直接看到转化率的广告投放（如在社交媒体买量）。 |
| **Negotiated Sourcing Costs** | 协商后的采购成本 | 供应链团队与代工厂砍价后降低的生产成本。 |
| **Reduction in Force (RIF)** | 裁员 | 美国高管最爱用的“体面黑话”，代替刺耳的 Layoff 或 Firing。 |

## 三、 管理层话术与高阶动词 (Management Phrasing & Action Verbs)

| 英文词汇 | 中文释义 | 商业语境与潜台词 |
| --- | --- | --- |
| **Differ Materially** | 产生重大/实质性差异 | 财报免责声明高频词。指实际结果与预测偏离极大。 |
| **Repurchases / Share Buybacks** | 股票回购 | 公司拿现金买回自己的股票并注销，减少市面流通股，推高股价以回馈股东。 |
| **Debt Pay-down** | 偿还债务 | 公司用赚来的自由现金流还债，降低杠杆风险。 |
| **Diversified Revenue Streams** | 多元化的营收流 | 收入来源多样化，不依赖单一爆款，抗风险能力强。 |
| **Revenue Pillars** | 营收支柱 | 能够独立撑起公司业绩大盘的核心大品类或子品牌。 |
| **Recalibrating** | 重新校准 / 调整 | 财报高情商黑话。通常暗示“收紧打折力度，优化促销节奏”。 |
| **Pull Back On** | 收紧 / 缩减 / 减少 | 例如 pull back on promotions，意味着减少打折让利。 |
| **Offset in Part By** | 部分被...抵消 | 汇报业绩时的万用句型（一好一坏同时发生）。例：销量增长被汇率劣势部分抵消。 |
| **Stabilize** | 稳住 / 企稳 | 潜台词：该业务之前在暴跌，现在的首要目标是止血，别再恶化了。 |
| **Anniversaried** | 基数效应满一周年 | 解释费用同比变化时的黑话。指去年的某项特殊花销/增长已经满了一年，不再对今年的同比数据产生异常影响。 |
| **The Shape of the Year** | 全年业绩走势分布 | 提前做好预期管理，比如预告业绩是“前高后低”还是“前低后高”。 |

## 四、 趋势、程度与数字单位表达 (Trends, Magnitudes & Numbers)

| 英文词汇 | 中文释义 | 商业语境与潜台词 |
| --- | --- | --- |
| **1 Million (1m)** | 100 万 | 财报最基础单位（如 128 million = 1.28亿）。 |
| **1 Billion (1b)** | 10 亿 | 跨国大公司的营收度量衡（如 1.5 billion = 15亿）。 |
| **Basis Points (bps)** | 基点 | 100 bps = 1%。专门用来精确表述百分比（如毛利率）的变动情况。 |
| **Year on Year (YoY)** | 同比 | 与去年同一时期对比（剔除季节性干扰）。 |
| **Consecutive** | 连续的 | 例：10th consecutive quarter（连续第10个季度）。 |
| **High Single Digits** | 高个位数 | 财报区间黑话。约等于 7% - 9%。(同理：Low single digits = 1%-3%) |
| **Headwind** | 逆风 / 不利因素 | 拖累业绩的宏观因素（如关税、通胀、汇率变动）。 |
| **Tailwind** | 顺风 / 利好因素 | 助推业绩的宏观利好。 |
| **Outsized Impact** | 巨大的 / 超出常规比例的影响 | 指某个举措或事件对报表产生了非常明显、不容忽视的冲击。 |
| **Outpace** | 跑赢 / 超过增速 | 优于 grow faster than 的高级商务词汇。 |
| **On a Reported Basis** | 按报告/账面基准 | 算上汇率波动等所有表面因素后，最终报表上呈现的真实数字。 |
